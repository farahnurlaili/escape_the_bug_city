🎮 Syntax Escape: The Runtime Riddle
Syntax Escape is an interactive 2D escape-room game built with Python and Pygame. It combines classic 2D platforming mechanics with Python programming logic. Players navigate through themed rooms, solve coding-based puzzles, and outsmart the "Runtime" to earn their freedom!

🧠 About the Project
This project was developed to demonstrate how Object-Oriented Programming (OOP) and state management can be applied to create scalable game engines. By moving away from procedural code to a BaseRoom class architecture, the game is designed to be easily extensible with new levels and mechanics.

🚀 Key Features
Modular Architecture: Uses a BaseRoom class to manage room lifecycles, making it easy to add new levels.

Gamified Learning: Puzzles require knowledge of Python syntax (loops, data types, logic).

Physics Engine: Custom gravity and jump logic for fluid character movement.

State Management: Tracks scores, lives, and timers across sessions to maintain game balance.

Retro Aesthetic: Designed with a classic pixel-art feel for an immersive retro experience.

🛠️ Tech Stack
Language: Python 3.11

Engine: Pygame 2.6.1

Design Pattern: State Machine & Object-Oriented Programming (OOP)

🎮 How to Play
Clone the repo: git clone [your-repository-url]

Install requirements: Ensure you have Pygame installed: pip install pygame

Run the game: Execute main.py to start the adventure.

Controls:

Arrow Keys: Move and Jump.

Mouse: Solve puzzles and navigate the UI.

ESC: Pause the game.
